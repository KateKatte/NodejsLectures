# Node.js course.

## Занятие 1.
- Что такое Node.js и с чем его едят.
- Модульная система.
- module, export, require(), global.
- Глобальные объекты.
- Встроенные модули.
- Работа с командной строкой и файловой системой.

## Занятие 2.
- Менеджеры пакетов.
- Node Package Manager(npm). 
- Зачем нужен Package.json и как его использовать.
- Разница между пакетом и модулем.
- Установка, удаление, обновление npm пакетов.
- Кастомные npm пакеты.

## Занятие 3.
- Обработка асинхронных операций.
- Ад колбэков.
- Promise.
- Async await.
- Generators.

## Занятие 4.
- TCP, HTTP, HTTPS.
- Простой TCP чат.
- Http/Https сервер с использованием встроенных node модулей.
- Http/Https сервер с использованием Express.js фреймворка. 

## Занятие 5.
- Веб фреймворки.
- MVC(Model View Controller).
- Express.js фреймворк.
- Промежуточные обработчики.
- Маршрутизация.
- Шаблонизаторы.
- Обработка ошибок.

## Занятие 6 - 7.
- Азы SQL.
- Азы NoSQL.
- SQL vs NoSQL.
- ORM vs ODM.
- MondoDB.
- MySQL.

## Занятие 8.
- Архитектурные шаблоны.
- CBA(Component Based Architecture)
- CBA vs MVC.
- Работа со сторонними api.
